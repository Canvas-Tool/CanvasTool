# 👉 Join the CanvasTool Community Discord!

CanvasTool is a browser-based study workflow helper for Canvas users, offering students a smoother way to navigate coursework, set up study tools, and access CanvasTool features through a simple license key system. CanvasTool is built to help students move through coursework workflows faster, with clear setup instructions and community support.
